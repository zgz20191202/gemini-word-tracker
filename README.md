# Gemini Word Tracker

A lightweight, privacy-focused Chrome Extension that tracks how many words you type into and read from the Gemini Web UI (`gemini.google.com`) on a daily basis. 

To make your usage stats more meaningful, the tracker provides a fun analogy by comparing your daily reading volume to the length of *Harry Potter and the Sorcerer's Stone* (approx. 76,944 words).

## Features
- **Dynamic Input Tracking:** Counts your words as you type them into the input box. If you type a prompt but haven't sent it yet, it still counts towards your daily total.
- **Dynamic Output Tracking:** Counts the words generated by Gemini in real-time as the response streams onto your screen.
- **Cross-Thread Accumulation:** Your daily word counts accumulate seamlessly even if you switch between different conversation threads or open new tabs.
- **Privacy First (No Data Retention):** This extension operates entirely within your browser using DOM observation. **No data is ever sent to external servers or databases.** All word counts are stored locally on your device using `chrome.storage.local`.
- **Unobtrusive UI:** Displays a clean, floating widget in the bottom right corner of the screen that supports dark mode. You can toggle the widget's visibility via the extension popup.

## How It Works
The extension relies purely on **DOM Observation** rather than network interception. This makes it highly resilient to API changes and ensures a high degree of privacy:
1. **Input Counting:** It listens to native browser `input` events within the chat text area. It tracks the maximum number of words you've typed in a single burst. Once the input box is cleared (e.g., when you hit "Send"), those words are permanently added to your daily total.
2. **Output Counting:** It uses a `MutationObserver` to watch for new text nodes being added to the chat interface. It accurately counts words as Gemini streams its answers, ignoring historical messages loaded when you switch threads.

## Installation (Developer Mode)
1. Clone or download this repository to your local machine.
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Enable **"Developer mode"** in the top right corner.
4. Click **"Load unpacked"** and select the folder containing this repository.
5. Go to [gemini.google.com](https://gemini.google.com/) and start chatting!

## Permissions Justification
- `storage`: Required to save your daily word count totals locally on your device so they persist across sessions.
- `*://gemini.google.com/*`: Required to inject the tracking scripts and floating widget directly into the Gemini interface.
