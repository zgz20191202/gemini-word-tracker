# Gemini Word Tracker

A lightweight, privacy-focused Chrome Extension that tracks your daily usage of the Gemini Web UI (`gemini.google.com`). It measures how many words you have asked and how many you have read from AI responses.

To provide meaningful context, the tracker compares your daily reading volume to the length of *Harry Potter and the Sorcerer's Stone* (approx. 76,944 words).

## Key Features
- **Smart Asked Count:** Counts words only after you send your message. It identifies "User Message" bubbles in the UI to ensure your typing effort is accurately captured once delivered.
- **Real-Time Read Count:** Dynamically counts words generated by Gemini as they appear on your screen.
- **Multilingual Support:** Accurately counts words in **English, Chinese, Japanese, and Korean**.
  - Latin-based languages are counted by word boundaries.
  - CJK (Chinese, Japanese, Korean) languages are counted by individual characters (the industry standard).
- **Cross-Thread Tracking:** Seamlessly aggregates your usage across different conversation threads and multiple browser tabs for the entire day.
- **Privacy & Security:** 
  - **No Data Retention:** This extension does not save your chat content. It only tracks numerical word counts.
  - **Local Only:** All data stays on your device in `chrome.storage.local`. Nothing is ever sent to a server.
- **User-Friendly UI:** A clean, floating widget that supports dark mode and can be toggled on/off via the extension popup menu.

## How It Works
The extension uses a high-performance **MutationObserver** to monitor the Gemini interface. It distinguishes between your message bubbles and Gemini's response bubbles using the page's DOM structure. This approach is more private and robust than intercepting network requests.

## Installation (Developer Mode)
1. Clone or download this repository.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **"Developer mode"**.
4. Click **"Load unpacked"** and select this project folder.
5. Refresh [gemini.google.com](https://gemini.google.com/) to start tracking.

## Technical Details
- **Tech Stack:** Vanilla JavaScript, CSS, HTML (Manifest V3).
- **Storage:** `chrome.storage.local` with daily keys (e.g., `2026-04-12`).
- **Performance:** Optimized to run silently in the background with minimal CPU impact.
